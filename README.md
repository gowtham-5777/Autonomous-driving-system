# Autonomous Driving Assistance System (ADAS)

AI-based perception and decision-support web application for real-time road video analysis.

## Modules

1. Lane Detection (YOLOP)
2. Vehicle & Object Detection (SSD MobileNetV2)
3. Traffic Sign Recognition (YOLOv5)
4. Traffic Signal Detection (CNN)
5. Semantic Segmentation (U-Net)
6. Decision Support (Rule-based)
